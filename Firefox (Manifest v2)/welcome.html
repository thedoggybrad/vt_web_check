Welcome to VirusTotal Website Security Check!

(I know this is annoying, I was just forced by browser vendors to flash this message)

This page just wants you to just be informed that when you click the add-on/extension icon on your web browser it will send the URL of your current tab to VirusTotal for it to check if the site is safe or not. Obviously, that's the sole function of this extension to forward the request to VirusTotal.
Sorry for popping this up, but this was just a requirement by browser vendors and it is not my full intention to have this thing.

For you to be sure that what I am stating here is all true, please visit the open-source code of this extension on https://github.com/thedoggybrad/vt_web_check

This popup was first required by Mozilla but if you are seeing this on a non-Mozilla browser I have also implemented this on other web browsers just in case the other browser vendors will require the same popup in the future.
